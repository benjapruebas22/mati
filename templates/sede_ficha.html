Criterios de analisis
Parametros V1
  • Ocupacion: 8-12 m2/persona = adecuado; <8 = insuficiente; 12-20 = amplio; >20 = sobredimensionado.
  • Ajuste potencial: se pondera disponibilidad de depositos como capacidad de reorganizacion operativa.
  • Densidad: hasta 2 pers/of = adecuada; 2-3 = intensivo; >3 = saturado.
  • Seguridad: vencen pronto (45d): 0 = ok; 1-2 = atencion; 3+ = intervencion.
  • Indice general interno: promedio de ocupacion + densidad + seguridad.
Detalle base de datos
  • sedes_metricas: m2_totales, personas, oficinas, depositos.
  • sedes_infra: salas_entrevistas, banios, espacios_comunes (y deposito de referencia).
  • seg_vencen: vencen_pronto (horizonte 45 dias).
