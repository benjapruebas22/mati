Instructivo Ergonomía

Guía técnica y operativa para carga en campo con cálculo automático.

Volver

Paso 1. Cargar Datos Base

  1. Seleccionar persona.
  2. Completar fecha de relevamiento y evaluador.
  3. Completar fecha de nacimiento (calcula edad automática).
  4. Completar horas diarias en PC.

Paso 2. Relevar Puesto

  1. Silla y escritorio.
  2. Soporte y altura del monitor.
  3. Espacio para piernas y ajuste recomendado.
  4. Uso de notebook (con o sin base).

Paso 3. Puntajes Actuales

ReglaPuntaje
Silla fija+2
Silla giratoria+1
Sin soporte monitor+1
Monitor bajo/alto+1
Espacio reducido / insuficiente+1 / +2
Notebook sin base+2
Horas PC ≥ 6+1
Promedio operativo = (punt_edad + punt_puesto + punt_salud) / 3. Si falta edad o salud: Pendiente.

Paso 4. Semáforo y Acción Automática

Bajo: 0-3 Medio: 4-6 Alto: 7+

La acción se calcula automáticamente: Condición adecuada Programado Urgente

Paso 5. Gestión del Caso

Flujo: Programado → Riesgo evaluado → Implementado → Verificado → Cerrado

Usar botones según corresponda: Reevaluar o Cerrar caso.

Alertas Automáticas

  • Edad 60+: alerta.
  • Edad 70+: crítica.
  • Horas PC > 6.
  • Puntaje puesto alto.
  • Caso urgente.

Auditoría y Trazabilidad

  • Fecha y hora de cada cambio.
  • Usuario/evaluador.
  • Campos modificados.
  • Snapshot completo del registro.
